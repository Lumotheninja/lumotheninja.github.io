Problem 24 - recursion

def lexigraph(pos, digits_list):
    if len(digits_list) == 1:
        return str(digits_list[0])
    factorial = math.factorial(len(digits_list)-1)
    index = pos//factorial
    first_digit = digits_list.pop(index)
    return str(first_digit) + lexigraph(pos%factorial, digits_list)

I really liked this question as it made me understand the very wonderful Totient generative function. I interpret is as such: if you are a prime number x, all your future multiples will have 1/x less, kinda like sieve of Erosthenes.

Problem 88 - a surprising DP

The key to solving this problem is to realize that:

  1. There is a trivial solution for every number x - 1,1,1,1,…2,x
  2. For any group of number (a,b,c), it is a solution for some x if you pad enough 1’s in front. e.g. 
    3*4 - (3+4) = 5
1*1*1*1*1*3*4 - (1+1+1+1+1+3+4) = 0

    After that, DP and you are done baby
